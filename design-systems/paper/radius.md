# Border Radius

| Token | Value | Default usage |
|---|---|---|
| base | 6px | Cards, modals, buttons, inputs, major surfaces |
| default | 6px | Badges, tooltips, dropdown items, code blocks |
| sm | 4px | Checkboxes, inline code, tiny elements, kbd |
| xs | 2px | Focus outlines, subtle rounding |
| full | 9999px | Pills, avatars, toggles, dot indicators |

## Rules

- 6px is the default radius across the product
- Never use arbitrary radius values outside this scale
- Radius must be consistent within each component family
- Cards, modals, buttons, inputs, and major containers all use 6px
- Small interactive elements (tags, code, checkboxes) use 4px
