# Lists

> Dependencies: `colors.md`

## Core Specs

- Item spacing: 8–16px vertical gap between list items
- Text: foreground at 90%
- Font: 18px (text-base), 28px line-height

## List Icons

- Size: 20x20px
- Prevent squishing: no shrink
- Spacing: 6px right margin between icon and text
- Active/featured icon: blue color
- Neutral icon: foreground at 50%

## Inactive / Disabled Items

Strikethrough text with foreground at 40% color on the list item.

## Pattern

Vertical flex list with 8–16px gap. Each item is a flex row with centered alignment — icon (20x20, no-shrink, 6px right margin) followed by a span of foreground-colored text at 90% opacity.
