# Icon Shapes

> Dependencies: `colors.md`, `radius.md`

## Core Specs

- Box sizing: border-box
- Icon must be perfectly centered (inline-flex, centered both axes)
- Circle: fully rounded (9999px)
- Rounded square: 6px radius (MD/LG/XL), 4px radius (XS/SM)

## Sizes

| Size | Container | Icon |
|---|---|---|
| XS | 24x24px | 14x14px |
| SM | 32x32px | 16x16px |
| MD | 40x40px | 20x20px |
| LG | 48x48px | 24x24px |
| XL | 56x56px | 28x28px |

## Color Variants

### Blue (Brand)
- Shape: circle
- Background: blue-300 at 15%
- Icon color: blue-500

### Gray
- Shape: circle
- Background: gray-100 (light) / gray-800 (dark)
- Icon color: foreground at 60%

### Danger
- Shape: circle
- Background: danger-soft
- Icon color: fg-danger

### Success
- Shape: circle
- Background: success-soft
- Icon color: fg-success

### Warning
- Shape: circle
- Background: warning-soft
- Icon color: fg-warning
